■Emoji Paletteについて
・CakeKtaiLibrary用のパレット
・docomoの基本絵文字
今のところCakeKtaiLibrary用のものしかありません。

■使い方
・nbmファイルをインストールする
・ctpファイルにphpを関連付ける or cakephp-netbeansプラグインをインストールする
・使いたい絵文字をパレットからエディタ側にドラッグ＆ドロップ
<?php $this->Ktai->emoji(0xe63e) // 太陽?>の形式のものが入力されます。
